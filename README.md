# ⚡ Pickleball Personality Test

A cyberpunk-themed personality assessment designed for **Pickleball** players. Discover your court archetype and match your soul with pro-player DNA.

---

## 🚀 Project Overview

This mini-game is a creative fusion of modern tech and sports culture, featuring:

- **AI-Generated Art:** Unique character avatars created via Generative AI to represent different playstyles.
- **AI Music:** Immersive background tracks generated to enhance the "Maniac Labs" laboratory atmosphere.
- **Frontend Logic:** A responsive web interface with dynamic color-theming (Neon Blue, Pink, and Green) that reacts to your results.

---

## 🛠️ Features

- **Neural Decoding:** Immersive loading sequences with randomized tech-jargon.
- **Dynamic UI:** Real-time edge-glow and panel shifts based on your result.
- **Tactical Export:** Built-in screenshot tool to generate a clean shareable card for social media.
